How Aiyo Studio Works

  1. Configure API Access:

    Generate an API key from OpenRouter and paste it into the configuration panel. Keys are saved locally in browser storage.

  2. Select a Model:

    Choose an AI model. We recommend google/gemini-2.5-flash because it is extremely fast and writes accurate code.

  3. Enter App Description:

    Type in your custom app idea or click one of the preset templates (e.g. Pomodoro, Budget Manager). Be detailed about layouts and buttons you want.

  4. Generate & Preview:

    Click "Generate App". When complete, interact with the app in the mobile phone mockup. Switch to "Source Code" to copy/download the file!

State Persistence Built-in: The generated code automatically saves state (like history, settings, logs) in the browser's localStorage, so your mock apps remain fully interactive even when run locally as a double-clicked file!
